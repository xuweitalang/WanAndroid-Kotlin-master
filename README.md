# WanAndroid-Kotlin
WanAndroid 是基于 Kotlin + MVP + RxJava + OkHttp 实现好用好看的学习阅读类客户端, 包括首页，项目，体系，干货，搜索，收藏，妹子等功能。

项目地址： https://github.com/xing16/WanAndroid-Kotlin

欢迎大佬们 Star ！
